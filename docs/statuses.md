# Integration statuses

| Status            | Meaning                                                                                                                                       |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `verified`        | Built or exercised against the recorded upstream version or commit, with automated evidence in this repository.                               |
| `community`       | Maintained here or by contributors, but not officially bundled by the upstream project.                                                       |
| `guide-only`      | Based on public documentation or a stable extension point; the private/current upstream source was not available for end-to-end verification. |
| `planned`         | Researched and suitable, but no complete integration exists yet.                                                                              |
| `blocked-private` | Implementation requires legitimate access to private/commercial source or maintainer cooperation.                                             |
| `deprecated`      | Kept for historical users but no longer recommended.                                                                                          |

Statuses describe the integration in this repository. They do not imply an
official partnership with the upstream maintainer.
